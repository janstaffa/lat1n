### types:

- number -> **numerus**: _0,I,II,II,IV,..._
- string -> **verbum**: _"salve mundi"_
- boolean -> **veritas**: true -> _verum_, false -> _falsus_
- array -> **ager**

### memory management:

- variable -> **variabilis**
- variable name cannot contain capital letters

```
var x = 10;
---------------------
Declaro variabilis numerus x.
Assigno X in x.
```

### operators:

- arithmetic:

  | symbol | word         |
  | ------ | ------------ |
  | +      | addere       |
  | -      | subtraho     |
  | /      | divido       |
  | \*     | multiplicare |

- logical:

  | symbol | word |
  | ------ | ---- |
  | &&     | et   |
  | \|\|   | aut  |
  | !      | non  |

- comparison:

  | symbol | word         |
  | ------ | ------------ |
  | ==     | aequalis est |
  | >      | major est    |
  | <      | minor est    |

### conditionals:

- if -> **si**
- else -> **aliud**

```
if (x = 10) {
    x = 11;
    var y = 5 - 3;
    x = y;
}
---------------------
Si x aequale X est quam:
'Assigno XI in x.
Declaro variablilis numberus y.
Assigno subtraho III ab V in y.
Assigno y in x.
'.
```
